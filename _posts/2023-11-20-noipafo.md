## day 1

T1 一眼出，想到对每个字符串处理出了最大的排列和最小的排列，然后一个字符串是否合法就可以等价于它的最小的排列严格小于其他字符串的最大的排列的最小值，一看嘛，区间最小值嘛，就 ST 表，然后

```cpp
string Min(int l, int r){
    int d=log2(r-l+1);
    return max(st[d][l], st[d][r-(1<<d)+1]);
}
```

Min 里面写 max，寄
